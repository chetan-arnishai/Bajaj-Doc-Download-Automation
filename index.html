How to Run / टूल कसे चालवायचे

  1. Enter the Claim / Intimation Number(s) — separate multiple by comma or space.
    क्लेम नंबर टाका — एकापेक्षा जास्त असल्यास comma किंवा space वापरा.
  2. Click ▶ Start — automation will launch the browser automatically.
    Start दाबा — browser आपोआप उघडेल.
  3. Solve the CAPTCHA in the browser window when it appears.
    Browser मध्ये CAPTCHA सोडवा.
  4. Enter OTP in the browser when prompted.
    OTP विचारल्यावर browser मध्ये टाका.
  5. Do NOT touch keyboard or mouse during automation.
    ऑटोमेशन सुरू असताना हात लावू नका.
  6. Files are saved to Desktop → Bajaj-doc-Automation folder.
    Files Desktop वरील Bajaj-doc-Automation फोल्डरमध्ये सेव्ह होतात.
📄 Multiple Claims: All claims run in parallel. Each claim gets its own folder.
सर्व claims एकाच वेळी process होतात. प्रत्येक claim चा स्वतःचा फोल्डर असतो.
NOTE: Do NOT close the app or browser during automation.
टीप: ऑटोमेशन सुरू असताना app किंवा browser बंद करू नका.
